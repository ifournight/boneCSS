Standard Form

Form elements receive some default style, add input, select, textarea with .form-control to give default basic style, use .form-group to group logical unit.
.form-control have width 100%, .form-group have a default margin.

Inline Form

When form added class .form-inline, all form group will be layouted inline, when screen width under 1screen-sm-width, inline form will fallback to standard form

Horizontal Form

When form added class .form-horizontal, label and control will aligned horizontally
